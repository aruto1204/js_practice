# 練習問題 1: 基本的なエクスポート・インポート

## 問題

基本的な数学関数を提供するモジュールを作成してください。

### 仕様

1. `math.js` ファイルを作成
   - `add(a, b)`: 2つの数値を足す
   - `multiply(a, b)`: 2つの数値を掛ける
   - `square(n)`: 数値を2乗する
   - 以上の関数を名前付きエクスポート

2. `main.js` ファイルを作成
   - `math.js` から関数をインポート
   - 各関数をテストして結果を表示

## 実行方法

```bash
node main.js
```

## 期待される出力

```
5 + 3 = 8
5 × 3 = 15
5 の2乗 = 25
```
