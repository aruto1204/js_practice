# 練習問題 2: デフォルトエクスポート

## 問題

ユーザー情報を管理する `User` クラスをデフォルトエクスポートで作成してください。

### 仕様

1. `User.js` ファイルを作成
   - `User` クラスを定義
   - コンストラクタで `name` と `email` を受け取る
   - `getInfo()` メソッドで「名前: {name}, メール: {email}」を返す
   - デフォルトエクスポート

2. `main.js` ファイルを作成
   - `User` クラスをインポート
   - 複数のユーザーインスタンスを作成
   - 各ユーザーの情報を表示

## 実行方法

```bash
node main.js
```

## 期待される出力

```
名前: 太郎, メール: taro@example.com
名前: 花子, メール: hanako@example.com
```
