# 練習問題 3: 名前付きエクスポートと混合エクスポート

## 問題

商品管理システムの基本モジュールを作成してください。

### 仕様

1. `Product.js` ファイルを作成
   - `Product` クラスをデフォルトエクスポート
     - コンストラクタで `name`, `price` を受け取る
     - `getInfo()` メソッドで商品情報を返す
   - `TAX_RATE` 定数（0.1）を名前付きエクスポート
   - `calculateTax(price)` 関数を名前付きエクスポート
   - `calculateTotal(price)` 関数を名前付きエクスポート

2. `main.js` ファイルを作成
   - すべてをインポートして使用
   - 商品を作成し、税込価格を計算して表示

## 実行方法

```bash
node main.js
```

## 期待される出力

```
商品: ノートPC, 価格: ¥100,000
税額: ¥10,000
税込合計: ¥110,000
```
